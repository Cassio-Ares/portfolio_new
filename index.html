Api de gestão de Academia com MySql

Uma API para o backend de um site de gestão de academia utilizando Express, Knex com MySQL2, bcrypt e JWT. A API gerencia alunos, planos e exercícios de forma eficiente, com segurança garantida por criptografia de senhas e autenticação JWT. A documentação é detalhada e intuitiva, facilitando a integração e o uso da API por desenvolvedores externos. Essa solução assegura uma gestão eficaz e segura das operações de academias
