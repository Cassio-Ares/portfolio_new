Api de gestão de Academia com MySql

Uma API para o backend de um site de gestão de academia utilizando Express, Knex com MySQL2, bcrypt e JWT. A API gerencia alunos, planos e exercícios de forma eficiente, com segurança garantida por criptografia de senhas e autenticação JWT. A documentação é detalhada e intuitiva, facilitando a integração e o uso da API por desenvolvedores externos. Essa solução assegura uma gestão eficaz e segura das operações de academias

Tecnologias

React JavaScript Express Knex MySql Jsonwebtoken DotEnv Bcrypt express-bearer-token Swagger

API de e-commerce com MongoDB

Uma API para o backend de e-commerce completa e segura usando MongoDB com Mongoose, bcrypt e JSON Web Token (JWT). A documentação foi elaborada com Swagger para uma integração fácil e intuitiva. No painel administrativo, é possível cadastrar e precificar produtos, além de gerenciar listas de compra. Para os clientes, a API proporciona uma experiência fluida com busca de produtos, gerenciamento de listas de compra, seleção de formas de pagamento e acompanhamento do status de envio.

Tecnologias

React JavaScript Express Nodemon MongoDB Mongoose Jsonwebtoken DotEnv Bcrypt Swagger
